# LilGrant Articles

Drop your article files here as `.md` or `.txt`.

## File naming convention

```
YYYY-MM-DD-slug-here.md
```

Example:
```
2026-04-25-how-to-get-financial-aid-as-international-student.md
2026-04-25-need-blind-schools-explained.md
```

## Frontmatter format (for .md files)

```markdown
---
title: "How to Get Financial Aid as an International Student"
slug: how-to-get-financial-aid-as-international-student
date: 2026-04-25
author: LilGrant Team
tags: [financial aid, international students, guide]
excerpt: "A step-by-step guide to finding and applying for financial aid as an international student in the US."
---

Article body goes here...
```

## When ready to publish

Just tell the assistant: "Push the articles to the system" and it will read all files in this folder and wire them into the app.
